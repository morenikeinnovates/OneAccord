# 🎯 OneAccord - Start Here

Welcome! OneAccord is a **complete, production-ready platform** for couples to have guided conversations. This file will get you oriented in 2 minutes.

---

## 🚀 What You're Getting

OneAccord is a fully built, fully documented platform featuring:

```
✨ User Authentication        (Magic links, no passwords)
✨ Couple Management          (Link with partners using codes)
✨ 10 Conversation Categories (Faith, Communication, Money, etc.)
✨ Guided Session Workflows   (3-step conversation process)
✨ AI Declarations            (Claude-powered couple keepsakes)
✨ Admin Dashboard            (Track users, couples, analytics)
✨ Responsive Design          (Works on mobile & desktop)
```

## 🚀 3-Step Setup

### Step 1: Environment (2 min)
```bash
cp .env.example .env.local
# Add your 3 Supabase keys to .env.local
```

### Step 2: Install (2 min)
```bash
npm install
```

### Step 3: Run (instant)
```bash
npm run dev
# Visit http://localhost:3000
```
